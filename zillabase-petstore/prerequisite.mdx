---
title: 'Prerequisite'
description: 'Prepare your local environment for running Zillabase'
---

Ensure that the following requirements are set up in your local environment.

#### 1. Homebrew

Make sure [Homebrew](https://brew.sh) is installed.

To verify its installation, run this command.

```bash
brew --version
```

#### 2. Java

Make sure [Java](https://www.oracle.com/java/technologies/downloads/) is installed.

To verify its installation, run this command.

```bash
java --version
```

#### 3. Docker

Make sure [Docker](https://www.docker.com/products/docker-desktop/) is installed and running.

To verify that Docker is running, run this command.

```bash
docker info
```
